# Ekran Görüntüleri

Bu klasör LAB-3 responsive tasarım ekran görüntülerini içerir.

## Nasıl Ekran Görüntüsü Alınır

### 1. Mobil (375px - iPhone SE)

1. `npm run dev` ile projeyi başlatın
2. Chrome DevTools'u açın (F12)
3. Device Toolbar'ı aktifleştirin (Ctrl+Shift+M)
4. "iPhone SE" veya "375px" seçin
5. Tam sayfa screenshot: DevTools → ... menüsü → "Capture full size screenshot"
6. `mobile.png` olarak kaydedin

### 2. Tablet (768px - iPad)

1. Chrome DevTools'da "iPad" veya "768px" seçin
2. Tam sayfa screenshot alın
3. `tablet.png` olarak kaydedin

### 3. Masaüstü (1440px)

1. Tarayıcıyı tam ekran yapın (1440px veya daha geniş)
2. DevTools'u kapatın
3. Tam sayfa screenshot alın
4. `desktop.png` olarak kaydedin

## Alternatif: Browser Extension

- **Fireshot** (Chrome/Firefox)
- **GoFullPage** (Chrome)
- **Awesome Screenshot** (Chrome/Firefox)

## Görüntü İsimleri

- `mobile.png` - Mobil görünüm (375px)
- `tablet.png` - Tablet görünüm (768px)
- `desktop.png` - Masaüstü görünüm (1440px)
